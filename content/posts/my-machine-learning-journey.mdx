I started learning machine learning about six months ago. I had zero background in statistics and barely knew Python. This post is an honest reflection on that journey — what clicked, what didn't, and what I'd tell a beginner today.

## Why I Started

Honestly? The hype. Everyone was talking about AI, and I felt like I was missing something. But once I started digging in, I found something genuinely fascinating — the idea that you can teach a computer to learn patterns from data without explicitly programming every rule.

## The Resources I Used

I went through a lot of resources. Here's what actually helped:

### Python First
Before touching ML, I made sure I understood Python well — especially:
- NumPy (array operations are everywhere in ML)
- Pandas (for handling datasets)
- Matplotlib (for visualizing what's going on)

### Mathematics
I won't lie — I avoided this at first. Then I hit a wall when I couldn't understand *why* gradient descent works. I ended up going back and learning:
- **Linear Algebra**: vectors, matrices, dot products
- **Calculus**: derivatives (for backpropagation)
- **Statistics**: mean, variance, distributions

You don't need to go deep — just enough to understand the intuition.

### Hands-On Projects
The biggest unlock was building things. My first project was a simple linear regression model predicting house prices. Then K-means clustering on the Iris dataset. Small, but the learning was real.

## The Algorithms I Understand Now

After months of study and practice, I can confidently explain and implement:

| Algorithm | Type | Use Case |
|---|---|---|
| Linear Regression | Supervised | Predicting continuous values |
| Logistic Regression | Supervised | Binary classification |
| Decision Trees | Supervised | Classification & regression |
| K-Means | Unsupervised | Clustering |
| K-Nearest Neighbors | Supervised | Classification |

## The Mistakes I Made

### 1. Skipping the Math
I tried to shortcut the math by just using Scikit-learn. It worked for a while, but when I tried to understand neural networks, I was completely lost. Learn the basics early.

### 2. Too Much Theory, Not Enough Code
I spent two weeks watching lectures before writing a single line of code. Flip this ratio. Code while you learn.

### 3. Comparing Myself to Researchers
ML research papers are intimidating. You don't need to read them as a beginner. Focus on applying algorithms to datasets and understanding why they work.

## What's Next

I'm currently diving into **neural networks and deep learning**. I'm implementing a simple feedforward network in NumPy from scratch — no PyTorch, no TensorFlow — just matrix multiplications and backpropagation by hand.

It's painful, but I'm learning more from this one project than from hours of watching tutorials.

I'll write about that next. Stay tuned!
